# أفكار التصميم - منصة معيار

## الفكرة المختارة: "Enterprise Dark Precision"

### حركة التصميم
Corporate Minimalism مع لمسات Glassmorphism — تصميم مؤسسي داكن يوحي بالدقة والاحترافية.

### المبادئ الأساسية
1. **الدقة الهندسية:** كل عنصر له غرض، لا زخرفة عشوائية
2. **التسلسل الهرمي البصري:** معلومات مرتبة بوضوح تام
3. **الكثافة الذكية:** لوحات تحكم مكثفة المعلومات دون فوضى
4. **الثقة المؤسسية:** ألوان وطباعة تعكس الجدية والموثوقية

### فلسفة الألوان
- **الخلفية الأساسية:** #0A0F1E (Navy عميق جداً — يوحي بالعمق والاحترافية)
- **الأزرق الملكي:** #1E40AF (للعناصر التفاعلية والتأكيد)
- **الـ Cyan:** #06B6D4 (للإضاءات والإحصاءات المهمة)
- **الزجاجي:** rgba(255,255,255,0.05) مع backdrop-blur للبطاقات
- **النص الأساسي:** #E2E8F0 (أبيض دافئ قابل للقراءة)

### نمط التخطيط
- Sidebar ثابت (280px) قابل للطي (64px)
- Topbar لاصق مع بحث وإشعارات وقائمة المستخدم
- شبكة لوحة تحكم: 12-عمود responsive
- تخطيط غير متماثل للصفحة الرئيسية

### العناصر المميزة
1. **بطاقات زجاجية:** border مضيء + backdrop-blur + gradient خفيف
2. **خطوط إضاءة:** خط أزرق رفيع في أعلى البطاقات المهمة
3. **مؤشرات الحالة:** نقاط ملونة متحركة للحالات المختلفة

### فلسفة التفاعل
- Hover: رفع البطاقة 4px مع تعميق الظل
- Click: scale(0.97) مع 160ms ease-out
- Transitions: 200ms cubic-bezier(0.23, 1, 0.32, 1)

### نظام الرسوم المتحركة
- دخول العناصر: opacity 0→1 + translateY 12px→0 مع stagger 50ms
- Loading skeletons: shimmer animation
- Sidebar collapse: 250ms smooth

### نظام الطباعة
- **العربية:** Cairo (400, 600, 700) — نظيفة ومقروءة
- **الإنجليزية:** Space Grotesk (400, 500, 600, 700) — تقنية وعصرية
- **الأحجام:** 48px/36px/24px/18px/14px/12px
